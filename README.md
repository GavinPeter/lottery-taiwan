# Lottery Taiwan
==============
### Usage

This is a normal project

1 . Install Module

```
npm install
```

2 . Start Server

```
npm start 
```


## Routes
### Format
```
{ 
	date : "101 年 01 月 01 號", //日期
	result : [01, 02, …], //中獎號碼
	special : 05 // 特別號
}
```
### Path
* ```/649``` - 大樂透最新號碼
* ```/638``` - 威力彩最新號碼
